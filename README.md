# Conexões Digitalmente Humanas: O Impacto das IAs nas Relações Pessoais 

## 📒 Descrição
O e-book de 23 páginas explora como a inteligência artificial está transformando as relações humanas, analisando os efeitos das tecnologias digitais na comunicação, trabalho, empatia e interação social.

## 🤖 Tecnologias Utilizadas
IA Generativa ChatGPT para criação e resumo do conteúdo;
IA Generativa Copilot para gerar as imagens;
Para ediçao do e-book- Canva.

## 🧐 Processo de Criação
O e-book foi criado em várias etapas.
Pedi ao chat GPT para me sugerir temas para um ebook sobre uso das IAS.
Após a escolha do tema, ainda no chat GPT, pedi para que ele criasse um ebook de no máximo 15 páginas.
Pedi que explicasse melhor alguns capítulos.
Depois do conteúdo pronto pedi a gestão de títulos.
Para geração das imagens pedi ao Copilot que gerasse com base no título.
No canva editei em formato livro.
